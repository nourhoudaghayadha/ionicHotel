First of all , Thanks for purchasing our Code . We guarantee you to support online as possible as we have . Follow the steps to fully functional this Ionic 6 App . If any problem occours , Please mail us .

Step 1

After purchasing Code, You will get a ZIP file . Extract this file with winzip / winrar .You will get a folder which has source code of app.

Step 2

Check in your machine / Computer that Node will be installed .If not please install it latest Version .

Step 3

Check in your machine / Computer that Angular 12/13 will be installed .Fire the following command to check Angular Version in node command prompt.

ng version

If not install please install by using following command.

npm install -g @angular/cli

Step 4

Check in your machine / Computer that Ionic 5/6 will be installed .Fire the following command to check Ionic Version in node command prompt.

ionic --version

If not install please install by using following command.

npm install -g @ionic/cli

Step 5

Open node command prompt & navigate project root folder where we extracted our source code . Fire following command.

npm install

It will install all packages which requires to our project. After installing packages ,fire following command in order to serve our app in browser .

ionic serve

Congratulations , your app will serve you in browser for you . You may edit as like you.
